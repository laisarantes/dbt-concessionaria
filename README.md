# Projeto de Engenharia de Dados - Concessionária

🚧 Status: Em desenvolvimento

Este projeto tem como objetivo simular um pipeline de Engenharia de Dados utilizando tecnologias amplamente empregadas no mercado.

## Tecnologias
- Python
- PostgreSQL
- dbt
- Apache Airflow (em implementação)
- Docker
- Pandas

## Etapas do projeto
- ✔️ Análise exploratória dos dados
- ✔️ Modelagem do banco
- ✔️ Transformações com dbt
- 🚧 Construção da camada de marts
- 🚧 Orquestração com Apache Airflow

Projeto desenvolvido para fins de estudo e prática em Engenharia de Dados.
